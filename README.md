<p>Expo is an open-source platform for making universal native apps that run on Android, iOS, and the web. It includes a universal runtime and libraries that let you build native apps by writing React and JavaScript. This repository is where the Expo client software is developed, and includes the client apps, modules, apps, and more. The <a href="https://github.com/expo/expo-cli">Expo CLI</a> repository contains the Expo development tools.</p>
